# 重要規則（永久牢記）

## 標注格式（鐵規則）
- ✅ 正確：`<@數字ID>`
- ❌ 錯誤：`<@名字>`、`@名字`、`<@username>`

## 討論接力規則
1. 每條回覆的**開頭**必須標注下一個發言者
2. 被標注者才能發言，沒被標注的請安靜
3. 用完標注後才能繼續說下一句，不能連續發兩條

## 黃金準則（所有頻道適用）
1. 單獨標註 → 只有被標記的 Bot 回覆
2. 標註 @everyone → 全部 Bot 一起回覆
3. Bot 標記 Bot → 被標記的 Bot 要回覆
4. 用戶回覆某 Bot → 只有那個 Bot 回覆
5. 標記格式 → 用 `<@ID>`，不用 `@名字`
6. 沒標註 → 安靜不說話

黃金準則適用於所有頻道，請牢記。

## 標注回覆條件（新規則）
只有當別人標注你在「文字的開頭或結尾」時，才需要回覆。
標注在中間 → 不需要回覆。
