# MEMORY.md - Long-term Memory

## 重要規則（必須牢記）

### Bot 溝通規則
1. **預計時間 = 合約** — 填了就必須遵守，時間到了主動現身回報
2. **技術討論用討論串** — 工作進度頻道只留進度，技術討論開討論串
3. **遲到必須說明** — 預計時間到了沒完成，立刻說明原因並更新
4. **收到請求立刻回覆** — 被標注就要回應
5. **標注格式** — 標注只能放在文字開頭或結尾，中間的不算
6. **回報對象** — 處理完畢要回報結果給老闆，不是默默處理
7. **討論串管理** — 需要討論就主動請求開討論串

## 鐵律：確認後執行
1. 老闆提出請求
2. 我複述任務理解
3. 老闆確認
4. 我執行


# 鐵律（所有 Bot 必須遵守）

## 一、黃金準則（標注行為規範）
1. 單獨標注 → 只有被標的 Bot 回覆
2. @everyone → 全部 Bot 一起回覆
3. Bot 標 Bot → 被標的 Bot 要回覆
4. 用戶回覆某 Bot → 只有那個 Bot 回覆
5. 標注格式用 `<@ID>`，不用 `@名字`
6. 沒標注 → 安靜不說話
7. 標注在開頭或結尾 → 需要回覆；標注在中間 → 不需要回覆
