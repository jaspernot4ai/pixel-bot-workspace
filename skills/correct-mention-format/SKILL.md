# 正確標注格式 skill

## 目標
讓所有 Bot 學會使用正確的 Discord 標注格式，避免使用錯誤的提及方式。

## Discord 標注格式規則

### ✅ 正確格式
在 Discord 中標注某人時，必須使用 **`<@ID>`** 格式，其中 ID 是該使用者的數字 ID。

正確範例：
- `<@1484830086350897152>` → 標注 Frontend
- `<@1484830707124535396>` → 標注 Backend
- `<@1484929073892429945>` → 標注 Data
- `<@1484929943417655296>` → 標注 Devops
- `<@1484930371307966535>` → 標注 QA
- `<@1484930784463949844>` → 標注 AI Engineer
- `<@1484931134713364480>` → 標注 SecOps
- `<@1484770562688680046>` → 標注 Home（轟爸）
- `<@803574097862721587>` → 標注人類（Ckweiiiii）

### ❌ 錯誤格式
- `<@名字>` → 例如 `<@Frontend>` 或 `<@pijuma-1>` — 這是錯誤的！
- `@名字` → 沒有包在 `<>` 裡 — 這也不是正確格式

## 常見錯誤
- ❌ `<@Frontend>` — 錯誤，應該用 `<@1484830086350897152>`
- ❌ `<@pijuma-1>` — 錯誤，這不是有效的 Discord ID
- ❌ `@everyone` — 可接受用於群體提及，但不是標注特定人的正確方式

## 記憶要點
1. **每個人都有獨特的數字 ID**，不是用戶名
2. **格式必須是 `<@數字ID>`**，中間沒有空格
3. **想知道某人 ID**，可以請管理者提供，或查看 Discord 開發者模式
